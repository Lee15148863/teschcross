# 🔔 滚动字幕公告系统

## 功能概述

主页顶部的可管理滚动字幕公告系统，管理员可以完全控制字幕的显示和样式。

## 功能特点

### 1. 滚动字幕显示
- ✅ 主页顶部滚动显示
- ✅ 双语支持（英语/爱尔兰语）
- ✅ 可开启/关闭
- ✅ 平滑滚动动画

### 2. 样式自定义
- 📝 字体大小（12-32px）
- 💪 字体粗细（Normal/Medium/Semi-Bold/Bold）
- 🎨 文字颜色（颜色选择器）
- 🎨 背景颜色（颜色选择器）
- ⚡ 滚动速度（10-60秒）

### 3. 管理后台
- 🔐 登录保护
- 📱 响应式设计
- 👁️ 实时预览
- 💾 自动保存到 localStorage

## 使用方法

### 访问管理后台

**方式1：隐藏入口**
- 在主页双击 Logo 图片
- 自动跳转到管理后台

**方式2：直接访问**
- 访问：`/announcement-admin.html`

### 登录凭证
- 用户名：`0876676466`
- 密码：`0870019999`

### 管理字幕

1. **开启/关闭字幕**
   - 使用顶部的开关按钮

2. **编辑文字内容**
   - 英语文本：主要显示文本
   - 爱尔兰语文本：切换语言时显示

3. **调整字体大小**
   - 范围：12-32px
   - 推荐：14-18px

4. **选择字体粗细**
   - Normal (400)
   - Medium (500) - 默认
   - Semi-Bold (600)
   - Bold (700)

5. **设置颜色**
   - 文字颜色：默认 #D4E157（品牌绿）
   - 背景颜色：默认 #000000（黑色）
   - 可使用颜色选择器或输入HEX值

6. **调整滚动速度**
   - 快速：10秒
   - 默认：20秒
   - 慢速：60秒

7. **实时预览**
   - 所有更改实时显示在预览区域
   - 保存前可以看到效果

8. **保存更改**
   - 点击"Save Changes"按钮
   - 设置立即生效

## 技术实现

### 文件结构
```
announcement-admin.html    # 管理后台页面
announcement-admin.js      # 管理后台逻辑
index.html                 # 主页（包含字幕显示）
script.js                  # 主页脚本（字幕加载）
styles.css                 # 样式（字幕动画）
```

### 数据存储
- 使用 `localStorage` 存储
- Key: `techcross_announcement`
- 数据格式：
```json
{
  "enabled": true,
  "textEn": "Welcome message...",
  "textGa": "Fáilte...",
  "fontSize": 15,
  "fontWeight": "500",
  "textColor": "#D4E157",
  "bgColor": "#000000",
  "scrollSpeed": 20
}
```

### 动画实现
- CSS `@keyframes` 动画
- 从右到左无限循环
- 速度可调（10-60秒）

## 最佳实践

1. **文字长度**
   - 建议：50-100个字符
   - 太短：滚动太快
   - 太长：阅读困难

2. **颜色对比**
   - 确保文字和背景有足够对比度
   - 默认组合（黄绿/黑色）对比度高

3. **滚动速度**
   - 根据文字长度调整
   - 短文本：15-20秒
   - 长文本：30-40秒

4. **更新频率**
   - 定期更新内容保持新鲜感
   - 重要通知及时发布

## 故障排除

### 问题：字幕不显示
**检查：**
1. 管理后台是否开启了字幕
2. 文字内容是否为空
3. 浏览器控制台是否有错误

### 问题：颜色不生效
**解决：**
1. 确保HEX颜色格式正确（#RRGGBB）
2. 点击"Save Changes"保存
3. 刷新主页

### 问题：滚动速度不对
**解决：**
1. 调整滚动速度滑块
2. 保存更改
3. 刷新主页查看效果
