# Tech Cross Repair Centre - 门户网站

现代化的3C零售店门户网站，专为移动设备优化，包含丰富的动态效果。

## 特性

- 📱 移动优先的响应式设计
- ✨ 丰富的动画和交互效果
- 🚀 纯静态HTML/CSS/JavaScript，无需构建
- 🎨 现代化的渐变色设计
- 💫 平滑滚动和页面过渡
- 🎯 优化的用户体验

## 本地预览

直接在浏览器中打开 `index.html` 文件即可预览。

## 部署到 GitHub Pages

1. 创建 GitHub 仓库
2. 推送代码到仓库
3. 在仓库设置中启用 GitHub Pages
4. 选择主分支作为源

## 部署到 Google Cloud

使用 Cloud Build 自动部署：

```bash
gcloud builds submit --config cloudbuild.yaml
```

## 自定义

- 修改 `index.html` 更新内容
- 修改 `styles.css` 调整样式和颜色
- 修改 `script.js` 添加或修改交互效果

## 浏览器支持

- Chrome (推荐)
- Safari
- Firefox
- Edge
- 移动端浏览器
